<h1 align="center">Sanatan AI</h1>

A beautifully crafted, animated web experience for Sanatan AI — a forward-looking assistant that blends modern technology with timeless wisdom, spirituality, and knowledge.

## 🌟 Overview

Sanatan AI is a polished, PWA-ready landing experience designed to introduce the project with elegance and clarity. The site highlights the mission of the platform, showcases its key features, and invites visitors to explore the experience in a modern, immersive way.

## ✨ Highlights

- Elegant and animated user experience
- Clean, modern presentation with immersive visuals
- Responsive layout for desktop and mobile
- PWA-ready structure for installable web app behavior
- Clear feature storytelling for the product vision

## 🚀 Live Demo

Visit the live app here:

- https://sanatan-ai.netlify.app

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- GSAP animations
- Progressive Web App support

## ▶️ Run Locally

1. Open the project folder in your editor.
2. Just open the [index.html](index.html) file in your preffered browser

## 📁 Project Structure

```text
.
├── assets/            # Images and branding assets
├── gsap/              # Animation libraries
├── index.html         # Main landing page
├── style.css          # Main stylesheet
├── script.js          # Interaction logic
├── site.webmanifest  # PWA manifest
└── README.md          # Project documentation
```
